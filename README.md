# Review Summerier

## Setup Enviornment Variables

```sh
export ASANA_TOKEN='your asana token'
export PROJECT_ID='your project id'
```

## Usage

### Using pipenv

#### Setup
```sh
pipenv install
```

#### Run
```sh
pipenv run python review_summrizer.py
```

### Using pip
#### Setup
```sh
pip install -r requirements.txt
```

#### Run
```sh
python review_summrizer.py
```
